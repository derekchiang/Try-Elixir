# Try Elixir

This is a project built with Elixir that uses Dynamo to serve web requests.

## Run

To run the site, make sure you have `mix` (it's shipped with Elixir) installed, then do:

1. Get dependencies with: `mix deps.get`

2. Run it: `mix server`
